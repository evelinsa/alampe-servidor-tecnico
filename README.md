# Alampe Servidor Técnico

Backend Node.js para o Alampe ERP.

## Endpoints

- `GET /api/health`
- `GET /api/mercadolivre?q=farol%20gol%20g6`
- `GET /api/aplicacao?q=farol%20gol%20g6`

## Rodar local

```bash
npm install
npm start
```

URL local: `http://localhost:3001`

## Variáveis

Veja `.env.example`.
